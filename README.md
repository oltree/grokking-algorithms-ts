Grokking Algorithms
About the Code
Implementation of examples and tasks from the book "Grokking Algorithms" in TypeScript.

About the Book
Algorithms are nothing more than step-by-step solutions to problems, and most of these problems have already been solved, tested, and verified by someone. You can, of course, dive into the profound philosophy of the brilliant Knuth, study lengthy volumes with proofs and justifications, but do you want to spend your time on that?
